This is a fix for the Adobe CC 2015 Sign In Required prompt. We have an enterprise license as such when your software become de-licensed please install this fix.

Please make sure no Adobe application(s) are running:
  • Adobe Photoshop CC 2015
  • Adobe Photoshop CC 2014
  • Adobe Premiere Pro CC 2015
  • Adobe Premiere Pro CC 2014
  • Adobe Illustrator CC 2015
  • Adobe Illustrator CC 2014
  • etc.


Note: When you click install this fix will be applied one time. Once the fix has been applied Managed Software Center will allow you to click install again, this is only needed if Adobe breaks again.
